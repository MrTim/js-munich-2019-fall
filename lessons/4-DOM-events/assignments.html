4.1) Change text

  • Write a function setText(elementId, text)
  • It receives a string with an HTML element id and a string with some text
  • Sets the element with the id to contain the text
  • Call it from your Chrome console with different values
  • Bonus: If the element with that id doesn't exist, alert with an error message

4.2) Swap elements

  • Write a function swapText(elementId1, elementId2)
  • It takes two element id strings as arguments
  • Swaps the two's text content! Afterwards, they should have exchanged their texts!
  • Add a simple site to try it out. Call the function inside the Chrome console.
  • Bonus: If one element doesn't exist, set the other's text to ":P"
  • Bonus: Add another function that takes three element ids

4.3) How are you?

  • Implement the "How are you?" app!
  • Create a HTML paragraph with the question How are you? and another paragraph answer.
  • You'll need to change the answer when a button was clicked, so add an ID!
  • Create two buttons with content Good and Bad
  • Add onClick event handlers
  • In the JS file create two functions that implement the event handlers
  • Find the answer paragraph and set its text content
  • REMEMBER: getElementById, textContent!

4.3) How are you?

  • Finish the "How are you" app.
  • Use CSS to style the good and bad buttons.
  • At the start of your script ask the user for her or his name with a prompt. Change the question to include the name of the user, e.g. "Hi Max, how are you?".
  • BONUS: When the user has clicked one of the buttons, hide the question paragraph. (Hint: set the hidden property of the element)
  • BONUS: Change the background color of the answer paragraph, e.g. green for good and red for bad. (Hint: look for the className property to set a class that you style with CSS)
