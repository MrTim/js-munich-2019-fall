  • Try: age >= 18

  • Update your function with the correct comparisons!

else

  • There's a way to simplify our code: else!
    • 
							function checkAge(age) {
								if (age >= 18) {
									return "adult";
								} else {
									return "child";
								}
							}

  • Comes after if
  • Covers all other cases

  • Comparisons are what we put inside a if(...)
  • Are used to compare two values
  • === (equals)
  • < (less than)
  • > (greater than)
  • <= (less than or equal to)
  • >= (greater than or equal to)
  • !== (not equal)

Exercise

  • You want to have a secret login to your website that only you can use
  • Write a function checkName(name)
  • It checks whether the name argument is the same as your name! (e.g. "Oliver")
  • If it is, return "permitted" => Show "permitted" in an alert
  • If not, return "denied" => Show "denied" in an alert.
    • 
							function checkName(name) {
								if (name === "Oliver") {
									return "permitted";
								} else {
									return "denied";
								}
							}
							alert(checkName(prompt("What's your name?")));

  • Comes after if
  • Covers all other cases
