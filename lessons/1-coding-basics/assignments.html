1.3) Adding Strings

  • Code a function named addStrings
  • Takes 2 arguments
  • Combine the 2 strings and store it in another variable
  • Then show the the stored variable in an alert box
  • Don't forget to make the function call.
  • BONUS: Add a third string argument

1.4) UserInfo

  • Code a function named showUserInfo
  • Takes at least 3 arguments: name, age, country, etc.
  • Combines the arguments to one greeting string
  • Then shows alert with the stored greeting
  • Call it with at least 2 different people
  • Don't forget to add comments!
  • BONUS: Before calling, ask for each value with prompts
