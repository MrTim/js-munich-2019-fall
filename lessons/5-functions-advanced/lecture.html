Scopes

  • between { and }, e.g. functions
  • variables are only visible in scope

					function f() {
						let inside = 2; // FUNCTION scope
					}
					f();
					alert(inside);

Scopes

Global variables (not in scope) are visible in the whole script!


					let global = 1;
					function f() {
						global = 2;
					}
					f();
					alert(global);

Different scopes!

						whatAmI = 1;  // FUNCTION SCOPE
					}
					f(whatAmI);           // GLOBAL
					alert(whatAmI);       // GLOBAL
