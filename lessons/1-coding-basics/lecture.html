Arguments to the rescue!

Functions can take so-called arguments

They are like boxes that can be filled-in later

Inside the function, you use them like variables


					function showDistance(speed, time) {
						let distance = speed * time;
						alert(distance);
					}

					showDistance(56, 7);
					showDistance(12, 3);
					showDistance(9, 88);
					showDistance(64, 90);

In JavaScript this is solved with
