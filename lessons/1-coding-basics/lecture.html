let vs const

  • Variables defined with const behave like let variables, except they cannot be reassigned.
  • const variables must be assigned a value when they are declared
    • 						
							const PI = 3.141592653589793;
							let pi = 3.141592653589793;
							PI = 3.14;	// This will give an error
							PI = PI + 10;	// This will also give an error
							pi = 3.14;	// This will work
							pi = pi + 10;	// This will also work

Local and Global Variables

  • When you declare a variable outside of any function, it is called a global variable, because it is available to any other code.
  • When you declare a variable within a function, it is called a local variable, because it is available only within that function

Example

					
						let carName = "Volvo";
						function car() {
						  alert(carName); // Here carName is a global variable
						}
						car();
					
				
					
						function car() {
							let carName = "Volvo";
						}
						car();
						alert(carName);

Undefined Variable

  • A variable declared using let statement with no assigned value specified has the value of undefined.
  • An attempt to access an undeclared variable results in a ReferenceError
  • Example: 
    							
								let a;
								alert(a);
								console.log(a);
