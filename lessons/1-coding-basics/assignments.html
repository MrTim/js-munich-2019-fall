1.1) Greetings

  • Write your code inside the greetings function.
  • Greet the user in an alert, then ask (prompt) for the user's name and store it in a variable.
  • Say "Nice to meet you, ..." (inserting the user's name).
  • Say goodbye with another alert.

1.2) Interview

  • Write your code inside the interview function.
  • Ask the user about their hobbies, home country, favorite color etc.
  • Use multiple prompts and variables.
  • At the end, give the user a summary of everything you have learnt about them!
  • "Hi Oliver! You are 28 years old and from Germany! You like music, programming and cooking and your favorite color is yellow ..."

1.3) User info

  • Codepen Starter: https://codepen.io/oliverbrehm/pen/bGGEqrL
  • Code a function named showUserInfo.
  • It takes at least 3 arguments: name, age, country, etc.
  • Combine the arguments to one greeting string.

1.3) User info

  • Show an alert with the stored greeting.
  • Call it with at least 2 different people.
  • Don't forget to add comments!
  • BONUS: Before calling, ask for each value with a prompt.
