2.3) Guessing Game - Basic

  • Write a guessing game where you prompt the player to guess the number you predefined from 0 to 10. For example let correctNumber = 7;
  • If the guessed number is correct, write the output to the browser using document.writeln(): YOU WIN!
  • If the guessed number is too high, write the output to the browser document.writeln():YOU LOST! TOO HIGH!
  • If the guessed number is too low, write the output to the browser document.writeln(): YOU LOST! TOO LOW!
  • Don't forget to write the css, and give the outcome a nice look.
  • Lastly don't forget to put in some helpful COMMENTS!
