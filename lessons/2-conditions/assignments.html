Number guessing

  • Write a guessing game where you prompt the player to guess the number you predefined from 0 to 10. For example let correctNumber = 7;
  • If the guessed number is correct, write YOU WIN!
  • If the guessed number is too high, write YOU LOST! TOO HIGH!
  • If the guessed number is too low, write YOU LOST! TOO LOW!
  • Hint: use if/else

Color guessing

  • Code a function named guessMyColor(correctColor)
  • Takes a correct color as an argument, e.g. "blue"
  • Asks the user to guess the color, then alerts "You guessed <color>"
  • If the guessed color is the same as the correct color, alert "Correct!"
  • Return the guessed color
  • Call the function with your favorite color
  • Hint: use switch/case

Guessing game
